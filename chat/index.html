Ask questions about the AsyncAPI specification, concepts, and tooling. Get answers grounded in the official docs, with sources you can verify.

Use it in your own tools — or run it locally

The AsyncAPI knowledge base speaks MCP. Wire it into your editor, install the Claude Code plugin, or clone the repo and run this whole chat on your machine.

Hosted MCP endpoint — .mcp.json 
{
  "mcpServers": {
    "asyncapi-knowledge": {
      "type": "http",
      "url": "https://asyncapi-knowledge-mcp.vercel.app/api/mcp"
    }
  }
}
Run locally via PyPI 
claude mcp add asyncapi-knowledge -- uvx asyncapi-knowledge-mcp
Claude Code plugin 
/plugin marketplace add derberg/asyncapi-knowledge-mcp
/plugin install asyncapi-knowledge@ai
Clone & run this chat locally 
git clone https://github.com/derberg/asyncapi-knowledge-mcp
./scripts/run-local.sh
